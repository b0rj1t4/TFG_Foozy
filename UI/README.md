# TFG UI README

## About

This project is part of my Final Degree Project (TFG). It focuses on building a user interface (UI) to support the main functionalities required for the TFG.

- **Ionic**: For building cross-platform mobile and web UIs.
- **Angular (Standalone Components)**: To structure the application with modern Angular features.
- **TypeScript**: To ensure type safety and improve code quality.
- **Tailwind CSS**: For efficient and responsive styling.
- **Vite**: For fast development and build tooling.

## Purpose

The UI is designed to provide a seamless experience for users interacting with the TFG application. It aims to be intuitive, responsive, and maintainable.

## Structure

- `src/`: Contains all UI components and logic.
- `public/`: Static assets.
- `README.md`: Project documentation.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```
